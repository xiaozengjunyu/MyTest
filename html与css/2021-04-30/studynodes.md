## 二、认识 HTML

### 1.(什么是 HTML )

 * HTML 是超文本标记语言 
 * HTML 就是显示浏览器页面的关键
### 2.( Web 服务器如何运作)

 * 首先我们知道 HTML 是超文本标记语言，要创建一个页面就需要创建一个超文本标记语言
 * 创建了一个 HTML 文件之后，将文件发送到 Web 服务器，再通过 Web 服务器
 * 最后Web服务器接收到了文件，当用户使用浏览器发送请求，Web 服务器接收请求将页面呈现给用户
### 3.(什么是浏览器页面结构)
 * 浏览器页面结构就是一种标记文本的方法，用成对的标记文本内容
 * 元素 = 标记开始 + 内容 + 标记结束
  
### 4.(认识块元素与内联元素)
 * 块元素独占一行，特立独行
 * 内联元素行内随波逐流
<p>块元素通常用来构建模块，而内联元素通常用来标记小段内容</p>
